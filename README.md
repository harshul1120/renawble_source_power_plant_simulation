# Solar-Wind Hybrid Power System Simulation (MATLAB/Simulink)

## Overview
This project presents a simulation of a hybrid renewable energy system that integrates solar photovoltaic (PV) and wind energy sources using MATLAB/Simulink. The objective is to analyze the performance, reliability, and efficiency of a combined power generation system under varying environmental conditions.

Hybrid energy systems improve reliability by combining multiple energy sources. Solar energy is available during daytime, while wind energy can generate power during non-sunny periods, ensuring continuous power supply.

---

## Key Features
- Modeling of a solar PV system using real panel parameters (voltage, current, temperature coefficients, resistances)
- Integration of wind energy generation into the system
- Simulation of dynamic environmental conditions (irradiance and temperature variations)
- Combined operation of solar and wind sources for continuous power supply
- Analysis of system performance and output stability

---

## System Components
- Solar PV Array (based on real module data)
- Wind Energy Conversion System
- Power electronic conversion stages
- Load and system output monitoring

---

## Tools & Technologies
- MATLAB
- Simulink
- Power System Simulation

---

## How to Run
1. Run `sunpower305.m` in MATLAB to initialize PV parameters.
2. Open `Main_generation.slx` in Simulink.
3. Run the simulation to observe system behavior.
